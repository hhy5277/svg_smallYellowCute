#使用svg绘制的图形

1）使用图形组合，有椭圆，多边形，曲线，直线，贝塞尔曲线，矩形等等；

2）使用蒙版mask

3）使用裁剪clip（这部分使用不多）

4）使用滤镜（filter）

5）使用user重用复制标签

6）使用阴影，使demo看起来更立体

7）使用animation动画（有些部分使用path路径设置动画），有些demo同时也有结合了animate.css动画框架设置动画

8）在用svg绘制图形前，需要理清各图形的画法和坐标设置，才能更得心应手
